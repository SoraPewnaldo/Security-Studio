# ENGINEERING RULES
**General**: Strict TypeScript, functional React components, no `any`.
**Code Style**: Small reusable components, feature-based folders, Zod validation, async/await only.
**Security**: Sanitize all inputs, validate APIs, Helmet enabled, rate limit, never trust client.
**Performance**: Lazy load routes, memoize expensive work, Web Workers for CPU-heavy tasks.
**Accessibility**: Keyboard nav, proper labels, visible focus, WCAG AA.
**Git**: feat: fix: docs: refactor: test: chore:
**PRs**: Add tests, update docs, log breaking changes.
**Docs**: Every feature requires README, Usage, Screenshots, Limitations.
