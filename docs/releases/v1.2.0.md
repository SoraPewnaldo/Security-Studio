# Release Notes — Security Studio v1.2.0

We are excited to release **Security Studio v1.2.0**! This milestone shifts Security Studio from a compilation of isolated utility calculators into a **guided cybersecurity assessment platform** with the introduction of **Security Playbooks** and an integrated **HTTP Client**.

---

## 🆕 Core Features

### 1. Security Playbooks (Assessment Engine)
Security Playbooks introduce guided step-by-step methodologies for common threat audits. Rather than switching between tools, tools are now embedded inline.

*   **Guided assessment panels**: Tells users *Why this step* is required and *What to look for* (suspicious threat checklists).
*   **Progress Indicators**: Features timeline steppers, completion checkboxes, progress bars, and state persistence (saved in browser `localStorage`).
*   **Built-in Playbooks**:
    *   🌐 **Domain Investigation** (OSINT WHOIS → DNS → ASN → TLS Certificate)
    *   🔐 **JWT Security Audit** (Decode → Signature verification)
    *   🍪 **Session Cookie Review** (Security flag audit)
    *   🐞 **Web Application Recon**

### 2. HTTP Client (API Auditor)
An offline API auditor built into the platform layout.
*   **Full Method Suite**: Supports GET, POST, PUT, DELETE, PATCH, HEAD, and OPTIONS.
*   **Custom Request Headers & Body Editors**.
*   **Automatic Content Sniffing**: Inspects response bodies and headers to prompt the analyst with smart pipelines (e.g. if a JWT is found, suggests decoding it in JWT Inspector; if a cookie header is set, suggests audits in the Cookie Analyzer).
*   **Local Network Support**: No outbound loops or localhost restrictions. Test local API endpoints directly.

---

## 🛡️ Bug Fixes & Refactoring

*   **CORS Mismatches**: Fixed backend CORS configurations to allow connection tunnels on development port `3000` (Vite's server).
*   **Auto-discovery registry validation**: Standardized export schemas. The http-client manifest now exports under the standard `manifest` object rather than a custom variable, restoring missing sidebar visibility.
*   **PEM Certificate pasting**: Added regex-based normalizers to clean corrupted string outputs before calling Peculiar/X509.
*   **Sidebar Navigation Polish**: Connected the inline Find bar to dispatch layout toggle events directly to the keyboard-shortcut command palette.
*   **Dependencies cleanup**: Cleaned up package.json structures and added missing binary utilities like `rimraf`.
